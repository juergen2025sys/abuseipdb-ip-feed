All Countries IPv4 Blocklist
A complete IPv4 blocklist containing all IP ranges from all countries worldwide (249 countries).
Designed for use with OPNsense, pfSense, or any firewall that supports URL-based IP tables.
URL
https://raw.githubusercontent.com/juergen2025sys/abuseipdb-ip-feed/refs/heads/main/all_countries_ipv4.txt
Stats

254,556 IP ranges (CIDR format)
249 countries covered
Source: country-ip-blocks.hackinggate.com

Use Case
Block all incoming WAN traffic by country. Since no inbound connections are needed, this completely eliminates attack surface from foreign IPs.
Works great in combination with:

Datacenter IP blocklists (AWS, Azure, Hetzner, etc.)
VPN provider blocklists (NordVPN, ProtonVPN, ExpressVPN, etc.)
TOR exit node blocklists
Threat intelligence feeds (FireHOL, CrowdSec, AbuseIPDB, etc.)
