

# NETSHIELD

NETSHIELD is a GitHub‑driven threat intelligence and IP blocklist framework that automatically aggregates,
scores, and exports malicious IPv4 infrastructure for firewall use.

The project uses GitHub Actions workflows to continuously update threat feeds, analyze ASN reputation,
detect Tor/VPN infrastructure, and export firewall‑ready blocklists.

---

## Main Features

### Automated Threat Intelligence
NETSHIELD automatically gathers and processes threat feeds using scheduled GitHub workflows.

Capabilities include:

- Aggregation of multiple threat intelligence feeds
- ASN reputation scoring
- Botnet and proxy detection
- Tor exit node monitoring
- False positive checks
- Feed health monitoring
- Confidence scoring and decay logic

---

## Generated Blocklists

The repository produces multiple blocklists designed for different firewall policies.

### Core Lists

| File | Description |
|-----|-------------|
| `combined_threat_blacklist_ipv4.txt` | Main aggregated threat blocklist |
| `active_blacklist_ipv4.txt` | Active malicious infrastructure |
| `blacklist_confidence40_ipv4.txt` | Medium confidence threat list |
| `bot_detector_blacklist_ipv4.txt` | Bot / automation detection IPs |
| `asn_blocklist_firewall.txt` | High‑risk ASN firewall blocks |

## Geo‑IP Datasets

NETSHIELD generates geographic IPv4 datasets.

### Global dataset

```
all_countries_ipv4.txt
```

Contains IPv4 CIDR ranges aggregated from all supported country lists.

### Continent datasets

```
continents/
```

Examples:

- `europe_ipv4.txt`
- `asia_ipv4.txt`
- `africa_ipv4.txt`
- `north_america_ipv4.txt`

### Country datasets

```
countries/
```

Structured by continent and country.

Example:

```
countries/europe/germany_ipv4.txt
countries/asia/japan_ipv4.txt
countries/africa/algeria_ipv4.txt
```

## Firewall Integration

The generated blocklists are compatible with common firewalls.

Supported systems include:

- OPNsense
- pfSense
- Linux nftables
- iptables
- router‑based blocking

Example firewall import:

```
URL Table Alias → combined_threat_blacklist_ipv4.txt
```
